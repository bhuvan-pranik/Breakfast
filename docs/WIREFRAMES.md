## Layout Measurement Reference

### Spacing
- Page padding (desktop): 32px
- Page padding (mobile): 16px
- Section gap: 32px (desktop), 24px (mobile)
- Card padding: 24px (desktop), 16px (mobile)
- Form field gap: 20px
- Button gap: 12px

### Component Sizes
- Header height: 60px
- Sidebar width: 240px
- Input height: 40px (base)
- Button height: 40px (base)
- Card border radius: 8px
- Modal border radius: 12px

### Typography in Layouts
- Page title: 32px (h1)
- Section title: 24px (h2)
- Card title: 18px (h4)
- Body text: 16px
- Small text: 14px
- Caption: 12px
